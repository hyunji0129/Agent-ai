분석 인사이트

5개 사이트 분석을 통해 도출한 포트폴리오 핵심 전략

01

해외 vs 국내 방향성

해외는 "경험 자체가 포트폴리오" — 인터랙션·3D 기술로 실력을 직접 증명.
국내는 "이력서형 포트폴리오" — 실무 프로젝트와 기술 스택을 명확히 정리하는 방식이 주류.

02

공통 성공 요소

① 첫 화면에서 명확한 정체성 전달 (직함·경력·강점)
② 실제 작업물 이미지로 역량 직접 증명
③ 빠른 로딩 + 반응형 지원
④ 연락처(CTA) 섹션의 명확한 배치

03

2026 기술 스택 트렌드

해외 고급: Three.js · WebGL · GSAP · Next.js
해외 중급: Webflow · Framer · Lenis Scroll
국내 표준: HTML·CSS·JS + GSAP or jQuery

04

레이아웃 선택 기준

카드형 그리드: 작업물 정리에 효율적 (이소영, Aino)
수직 섹션형: 스토리텔링에 효과적 (Juan Mora)
몰입형 풀스크린: 강렬한 첫인상 (Bruno Simon)

05

신입 퍼블리셔 벤치마킹 순서

1단계: 박세희 — 기본기 (HTML·CSS·JS 구조 파악)
2단계: 이소영 — 완성도 (GSAP·SCSS 적용)
3단계: Juan Mora — 방향성 (모션·인터랙션 고도화)

06

무료 배포 팁

GitHub Pages: 정적 사이트 무료 배포 (박세희)
Vercel/Netlify: Next.js·React 프로젝트 무료 배포
Webflow: 노코드로 고퀄리티 사이트 제작 가능

📌 포트폴리오 제작 핵심 체크리스트

✅ 정체성
첫 화면에서 직함·경력·강점을 3초 안에 전달
✅ 작업물
실제 프로젝트 이미지로 역량을 직접 증명
✅ 기술
사용 기술 스택을 명확히 명시 (채용담당자 필수 확인)
✅ 반응형
모바일·태블릿·PC 모든 해상도 대응 필수
✅ 속도
이미지 최적화·경량 코드로 3초 이내 로딩 목표
✅ CTA
연락처·이메일·SNS 링크를 눈에 잘 띄게 배치
